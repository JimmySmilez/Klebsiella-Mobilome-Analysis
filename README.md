# Klebsiella Mobilome Analysis
## 🦠Introduction
*Klebsiella pneumoniae* is classified by WHO as a critical priority pathogen. This is because of its high virulence, morbidity and mortality rate. It is also becoming increasingly multi-drug resistant. This project presents a detailed analysis of the WGS data of a multi-drug resistant *K. pneumoniae* isolate.  
## 📊Data Acquisition
The raw reads that were utilized in this project were obtained from NCBI SRA with accession number SRR32280978. The study was originally published by Byarugaba et al. 2023 (https://www.mdpi.com/2076-0817/12/11/1334/).
